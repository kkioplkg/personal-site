# Personal Site

这是一个纯静态的个人网站示例，已经放在 `personal-site/` 目录下。

## 文件结构

- `index.html`：主页内容
- `styles.css`：样式文件
- `script.js`：页脚年份和回到顶部按钮

## 本地预览

如果你装了 Python，可以在当前目录运行：

```bash
cd personal-site
python3 -m http.server 8080
```

然后打开 `http://localhost:8080`
